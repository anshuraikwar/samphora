# Virtual Gallery Prototype

A React web app prototype featuring a virtual gallery with seamless hover interactions and smooth page transitions, while maintaining keyboard accessibility.

## Live Demo

[View the project](https://anshuraikwar.github.io/samphora/)

## Features

- React-powered virtual gallery experience
- Smooth hover effects and page transitions
- Keyboard-accessible navigation and focus management
- Responsive gallery UI with interactive content details
- Built with React, JSX, and CSS

## How to run

1. Install dependencies with `npm install`
2. Start the app with `npm start`
3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Built with

- React
- JavaScript
- JSX
- CSS
